
GPIO
    GPIOA
        Pins 0-2:   extra pins
        Pin  3:     WS2812B output
        Pin  4:     audio output
        Pin  5:     debug LED
        Pins 13-14: programming header
    GPIOB
        Pins 0-7:  Keypad
        Pins 8-14: Sensors
    GPIOC 
        Pins 0-13: HUB75
        Cannot use free pins in "output" mode (value will be overridden) (HUB75)

Timers
    TIM 2
        PRE = 18; (WS2812B)
        ARR =  2; (WS2812B)
        Channel 4 (CC4) used (WS2812B)
    TIM 3 
        PRE = 0; (HUB75)
        ARR = 0xff; (HUB75)
        Channel 1 (CC1) used (HUB75)

DMA
    DMA1_CHANNEL2 (WS2812B)
    DMA1_CHANNEL4 (HUB75)

SRAM
    HUB75
        (#Columns + #ExtraColumns) * (#Rows / #RowsPerCycle) * #BytesPerPixel
        (64 + 3) * (32 / 2)  * 2  = 2144 bytes
    WS2812B
        3 * num_leds bytes

FLASH
    HUB75
        Fonts (unknown)
